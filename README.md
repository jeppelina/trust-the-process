# Burner Bash (Working Title)

A dark humor RPG inspired by West of Loathing and South Park: The Stick of Truth, set in the world of Burning Man culture, spiritual retreats, and new-age self-discovery.

## Premise

You are Steve. You work in accounting. Your wife went to an Ayahuasca retreat and never came back — not physically, but spiritually. She's lost in hippie land, and you have to go get her. Your journey takes you through yoga camps, healing courses, fire circles, tantra workshops, men's work retreats, and eventually Burning Man itself.

To survive, you'll need to acquire skills, collect absurd items, and level up your imaginary spiritual insights — not because any of it is real, but because everyone else thinks it is.

## Project Structure

```
docs/           — All written documentation and design
  game-design/  — Mechanics, systems, balance, GDD
  lore/         — World bible, factions, characters, quests, items, dialogue
  art-direction/— Style guides, tone references, visual direction
  planning/     — Roadmaps, task tracking, sprint plans, decisions log

game/           — Game engine project (TBD: Godot / RPG Maker / GameMaker)
  src/          — Source code
  assets/       — In-engine assets (sprites, audio, maps, UI, fonts)
  scenes/       — Scene/level files
  scripts/      — Game scripts organized by system

art/            — Source art files (pre-export)
  concepts/     — Concept art and sketches
  references/   — Mood boards, style references
  sprites/      — Character, enemy, boss, NPC, item sprites
  ui/           — Menu designs, HUD elements
  maps/         — Zone/area map designs
  environments/ — Background art, tileset sources

audio/          — Source audio files
  music/        — Background music tracks
  sfx/          — Sound effects
  voice/        — Voice lines (if any)

scripts/        — Build scripts, tooling, utilities
```
