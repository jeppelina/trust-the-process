# MVP Scope

## Goal
A playable vertical slice that demonstrates the core loop: explore zone, talk to NPCs, fight enemies, collect items, follow the wife's trail, beat a boss.

## MVP Contains

### One Complete Zone: The Healing Arts Center
Why this zone: It's mid-pipeline (not too gentle, not too extreme), has the richest item potential, and the enemy variety is great.

### Minimum Features
- [ ] Explorable map with interactable NPCs (5-8 NPCs)
- [ ] 3-4 enemy types with unique attacks and joke descriptions
- [ ] 1 boss fight (The Head Healer / Crystal Empress / TBD)
- [ ] Turn-based combat with Steve's basic accounting skills
- [ ] Inventory system with 8-10 collectible items
- [ ] Dialogue system with branching (at least for key NPCs)
- [ ] 1 main quest (find clue about the wife)
- [ ] 1-2 side quests
- [ ] Insight/XP system — gain insight from completing encounters
- [ ] Basic zone hub showing where other zones will be (locked)

### Explicitly NOT in MVP
- Multiple zones
- Full skill tree
- Save/load system (nice to have but not critical for demo)
- Music/sound (placeholder OK)
- Polished art (placeholder OK)
- Party members

## Success Criteria
1. Someone can play it for 20-30 minutes
2. They laugh at least 3 times
3. The core loop is clear: explore, talk, fight, collect, progress
4. It communicates the tone and promise of the full game
