# Lore & World Building

This is the world bible for Burner Bash. All narrative content lives here.

## Subfolders

### world/
The overarching setting — how this universe works, its rules, its tone. The spiritual-industrial complex as a living, breathing ecosystem.

### factions/
Each zone/camp is controlled by a faction. Document their beliefs, hierarchy, aesthetic, and how they interact with other factions.

### characters/
Steve, his wife, major NPCs, bosses, recurring characters. Include character sheets with personality, backstory, abilities, and key dialogue beats.

### quests/
Main quest line (finding the wife) and side quests per zone. Include quest flow, choices, rewards, and punchlines.

### items/
Every item in the game — equipment, consumables, quest items, joke items. Include name, description (the funny part), stats, and where it's found.

### dialogue/
Dialogue scripts organized by zone and NPC. Include branching options where relevant.

## Tone Guide

- Satire from familiarity, not cruelty — we know this world
- The mundane (Steve) clashing with the mystical (everyone else) is the core engine
- Dark humor is welcome but the joke should punch at the absurdity, not at vulnerable people
- Reference real trends but don't name real people/brands
- Every NPC genuinely believes what they're doing — that's what makes it funny
